GAME HISTORY

-> Get user stats
curl -X GET http://localhost:3000/user/stats -H "x-user-id: 1"

-> Get user games with pagination
curl -X GET "http://localhost:3000/user/games?page=1&per_page=5" -H "x-user-id: 1"

-> Get user tournaments with pagination
curl -X GET "http://localhost:3000/user/tournaments?page=1&per_page=5" -H "x-user-id: 1"

-> Get game details by ID
curl -X GET http://localhost:3000/games/classic_1 -H "x-user-id: 1"

-> Get tournament details by ID
curl -X GET http://localhost:3000/tournaments/tourn_1 -H "x-user-id: 1"

CUSTOM GAMES
-> Create custom game
curl -X POST http://localhost:3000/games/create \
  -H "x-user-id: 10" \
  -H "Content-Type: application/json" \
  -d '{
    "numberOfPlayers": 4,
    "invitedPlayerIds": [11, 12, 13]
  }'

-> Accept game invite
curl -X POST http://localhost:3000/games/GAME_ID/join -H "x-user-id: 11"

-> Decline Game Invite
curl -X DELETE http://localhost:3000/games/GAME_ID/decline -H "x-user-id: 12"

WEBHOOK GAME RESULT

-> P1 queue
curl -X POST http://localhost:3000/queue/join -H "x-user-id: 500"

-> P2 queue
curl -X POST http://localhost:3000/queue/join -H "x-user-id: 501"
